SportSee

Disclaimer: This repository houses a project undertaken as part of my "JavaScript React Developer" training program with OpenClassrooms. Upon the conclusion of the program on September 20th, 2023, the project will no longer be updated or maintained.

Overview

For this project, I had to build a sport analytics dashboard. The dashboard is a web application that allows users to visualize data about their sport activities.

Local setup

For the frontend of this project I used PNPM as a package manager. If you don't have it installed, you can install it with the following command:

npm install -g pnpm

Frontend

  1. Clone the repository
git clone https://github.com/alexperronnet/openclassrooms-12-sportsee.git
  1. Navigate to the project folder
cd openclassrooms-12-sportsee
  1. Install dependencies
pnpm install
  1. Start the development server
pnpm run dev
  1. Build the project
pnpm run build # Use real api
# or
pnpm run build:staging # Use mock api
